# Звуки игры

Кинь сюда mp3-файлы с **ровно такими именами** — игра подхватит их сама.
Без файлов игра тоже работает (просто молча).

| Файл          | Когда играет                                   |
|---------------|------------------------------------------------|
| `kick.mp3`    | удар по мячу (каждое набивание)                |
| `goal.mp3`    | забил вратарю — «ГОООЛ» + стадион              |
| `whistle.mp3` | уронил мяч — свист толпы                        |
| `life.mp3`    | +1 жизнь за каждые 100 очков (приятный звук)    |
| `save.mp3`    | вратарь поймал мяч (необязательно)             |

Формат: mp3, лучше короткие (kick — совсем короткий, до ~0.3 сек).
