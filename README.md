# ⚽ Набивалка

Мобильная HTML5-игра (вертикальная) в стиле 16-битной Сеги.

**Суть:** набивай мяч (тап = удар), не урони его. Фон с гор и облаками
прокручивается parallax'ом, будто персонаж идёт. Каждые 20 ударов —
пенальти: целься в ворота и бей, вратарь ловит 50/50.

## Правила
- 1 удар = **+1 очко**
- Гол вратарю = **+50 очков** (звук «гооол» + стадион)
- Уронил мяч = **−1 жизнь** (свист толпы), всего **3 жизни**
- Каждые **100 очков** = **+1 жизнь**

## Управление
- Набивание: **тап в любом месте** = удар по мячу
- Пенальти: **тап по воротам** (целишься) → **тап** (бьёшь)

## Как запустить
Открой `index.html` через локальный сервер (звуки требуют http, не file://):
```
cd football
python3 -m http.server 8080
# открой http://localhost:8080
```

## Звуки
mp3-файлы кладутся в `assets/sounds/` — см. `assets/sounds/README.md`.

## Структура
```
index.html        — разметка + экраны
css/style.css     — ретро-стиль
js/game.js        — вся логика игры
assets/sounds/    — звуки (mp3)
```
